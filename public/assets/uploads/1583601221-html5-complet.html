Dans le header d'une page web, vous pouvez trouver :

  • Un logo, avec un lien vers la page d'accueil
  • Le nom de la société/titre du site (ex: Le logo de Nike n'affiche pas le nom de l'entreprise)
  • Un bandeau publicitaire (lien interne ou externe)
  • Une balise <nav> contenant un menu principal (votre menu utilisera les balise <ul> et <li>
  • Un slogan ou descriptif court (ex : profession, spécialité...)
La balise <nav> peut également être placée en-dehors du <header> pour des raisons de webdesign

Titre <h1> décrivant toutes les sections de la balise <main> (optionnel)

Partie 1 : Titre de section

Le contenu de la balise <main> est découpé en sections.

Le W3C recommande que chaque section contienne un titre de type <h2> (ou <h1> s'il n'y a pas de titre principal pour toutes les sections).

Une section peut contenir des contenus autonomes : <article>

Un article est utilisé pour présenter un extrait d'une autre page

Le texte qui suit le titre d'un article est un "extrait" de la page présentant l'intégralité du contenu de cet article. L'extrait est généralement accompagné d'une image d'illustration...

Un <article> peut avoir son propre en-tête (élément <header>)

On utilise les articles dans les pages dynamiques en PHP

C'est une boucle PHP qui envoie une requête à la base de données afin d'afficher (par exemple) tous les contenus de type "fiche de cuisine" (ou "voiture d'occasion" ou même "réalisation" pour un portfolio).

On peut également trouver un <footer> dans un <article>

Quand peut-on utiliser un div ?

Il n'est pas interdit d'utiliser la balise <div>. On l'utilise lorsque qu'aucune balise sémantique ne permet de placer un élément de type 'inline' dans un élément de type 'block'.
Il est parfois plus avantageux d'utiliser une balise <div> plutôt que de modifier la propriété CSS 'display' d'un élément.

Balise <aside> dans un article

Même dans un <article>, la balise <aside> représente un contenu secondaire (complémentaire).

La balise <q> permet de mettre en forme une citation (on utilise <aside> autour de la citation). C'est une balise sémantique utilisée pour le terme "quote" (citation).

Quelle utilisation pour <aside> ?

Le W3C recommande d'utiliser la balise <aside> pour présenter du contenu "secondaire" de type :

  • barre latérale
  • citation
  • publicité
  • slider
  • groupes d'éléments de type <nav> (une menu contextuel - uniquement présent sur certaines pages)

Une 2ème façon d'utiliser <aside>

L'élément <aside> peut également être utilisé à l'intérieur d'une balise <article> pour présenter du contenu complémentaire à l'article.

  • sidebar de gauche avec un fond rouge
  • sidebar de droite avec un fond rouge

Dans le pied de page vous pouvez créer plusieurs "boîtes" (des <div> par exemple) pour créer des colonnes.

C'est dans le <footer> qu'on place généralement les informations de copyright et le lien vers les mentions légales obligatoires.
